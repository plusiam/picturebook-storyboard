# 📖 나의 그림책 스토리보드 확장판

> 남부초등학교 · 6학년 도덕 1단원 「나는 꿈 요리사」 × IB 학습자상 그림책 프로젝트

## 🔗 바로가기

**[👉 스토리보드 열기](https://plusiam.github.io/picturebook-storyboard/)**

## 📋 개요

학생들이 인쇄물 스토리보드에 작성한 내용을 온라인으로 정리하고, PDF·JSON으로 내보내기할 수 있는 웹 도구입니다.

## ✨ 주요 기능

- **동적 장면 관리**: 장면 추가·삭제·순서 변경·타입 변경
- **고정 페이지**: 표지(P1), 빛나는 순간(클라이맥스), 작가소개 — 삭제 불가
- **그림판**: 각 장면마다 펜·지우개·색상·굵기 + 사진 업로드
- **IB 학습자상**: 10가지 학습자상 선택 및 자동 연동
- **진행률 바**: 작성 현황 실시간 표시
- **PDF 내보내기**: A4, 6장씩 자동 페이지 분할
- **JSON 내보내기/불러오기**: 그림 포함/미포함 선택, 드래그앤드롭 불러오기
- **임시 저장**: localStorage 기반 자동 복원

## 🛠 기술 스택

- Vanilla HTML/CSS/JS (단일 파일)
- html2pdf.js (PDF 생성)
- Google Fonts (Noto Sans KR, Gaegu)
- GitHub Pages 배포
